# **App Name**: DataLens Portfolio

## Core Features:

- Profile Showcase: Display professional details, including education, experience, and skills.
- Project Portfolio: Showcase completed projects with descriptions, technologies used, and links to repositories (e.g., GitHub).
- Skills Visualization: Present skills using charts or graphs for a quick overview of proficiencies.
- Achievement Timeline: Visually display a timeline of accomplishments, certifications, and career milestones.
- Recommendation Spotlight: Display recommendations received from colleagues and mentors.
- Contact Form with AI assistant: A contact form with a virtual AI tool to answer any basic questions from the website visitors.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust, intelligence, and professionalism.
- Background color: Light gray (#F0F0F0), subtly desaturated from the primary, offering a clean and modern backdrop.
- Accent color: Soft violet (#A78BFA), analogous to the deep blue, highlighting important interactive elements.
- Headline font: 'Poppins', a geometric sans-serif, offering a modern, fashionable, avant-garde look suitable for short amounts of text
- Body font: 'PT Sans', a humanist sans-serif font that combines a modern look and a little warmth or personality; particularly useful in combination with Poppins as a headline font
- Simple, professional icons to represent skills, achievements, and contact information.
- Clean, responsive layout with clear sections for easy navigation.
